## Accounting

### Requirements
* Node v16+
* NPM v7+

### Setup
Prerequisites:

* Installed NPM
* Installed postgres
```text
# Setup packages
npm i

# Ensure you have `.env` with properly configured variables.
npm run migrate:run

# Run dev mode
npm run start:dev
```

### Run app
* install packages: `npm install`
* build application : `npm run build`
* run: `npm run start`
* run watching: `npm run start:dev`
